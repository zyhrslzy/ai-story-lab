# AI 结局质量评测

## 目的

本评测用于判断 AI 根据固定故事背景和两次用户选择生成的结局是否达到可展示标准。评测对象包括结局标题、正文、人格关键词和分享文案，重点评分正文内容。

## 评分规则

- 每个维度使用 1～5 分，分数越高表示质量越好。
- 1、3、5 分采用下方的锚定标准；2 分表示表现介于 1 分与 3 分之间，4 分表示介于 3 分与 5 分之间。
- 四个维度等权，总分为四项分数的算术平均值，保留一位小数。
- 任一维度低于 3 分，测试案例即不通过，不因平均分较高而抵消。
- 四项均不低于 3 分时，案例通过。

## 评测维度

### 剧情连贯性

判断结局是否承接既有故事背景和选择路径，事件之间是否具有清晰的因果与时间关系。

| 分数 | 锚定标准 |
| --- | --- |
| 1 | 与前文明显断裂，出现无法解释的场景、事件或因果矛盾，难以理解发生了什么。 |
| 3 | 基本承接前文，主要事件可以理解，但存在轻微跳跃、信息缺口或转折生硬。 |
| 5 | 自然承接前文与两次选择，因果和时间关系清楚，转折顺畅且没有多余冲突。 |

### 角色一致性

判断角色的身份、动机、关系、语气和行为是否与固定设定及此前剧情一致。

| 分数 | 锚定标准 |
| --- | --- |
| 1 | 角色身份、关系或核心动机发生明显冲突，行为无法由已有设定解释。 |
| 3 | 角色核心设定保持一致，但个别语气、情绪或行为略显突兀。 |
| 5 | 角色身份、动机、语气和行为始终一致，并在结局中产生可信的发展。 |

### 选择差异性

判断结局是否真实反映两次选择，而不是仅替换少量词语；不同路径应在事件、关系、情绪或主题结果上形成可感知差异。

| 分数 | 锚定标准 |
| --- | --- |
| 1 | 选择几乎没有影响，结局可直接套用于其他路径，或与用户选择相矛盾。 |
| 3 | 至少一个选择得到明确回应，不同路径存在可识别差异，但另一选择影响较弱或停留在表面。 |
| 5 | 两次选择都产生清晰、自然且有意义的后果，当前结局不能无修改地套用于其他路径。 |

### 结局合理性

判断结局是否在当前世界观和篇幅内成立，是否完成情绪收束，并与都市奇幻治愈的产品基调一致。

| 分数 | 锚定标准 |
| --- | --- |
| 1 | 结局依赖未铺垫的关键设定或强行反转，无法解释主要冲突，或明显偏离治愈基调。 |
| 3 | 主要冲突得到基本收束，结局可以成立，但情绪回报、铺垫呼应或主题表达较弱。 |
| 5 | 结局在设定内可信，回应关键铺垫与选择，完成清晰而克制的情绪收束，并保留适当余韵。 |
